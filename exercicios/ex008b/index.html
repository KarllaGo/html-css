Outras formatações

Código-fonte / Pré-formatação

A tag <code> ela é usada para indicar o navegador que se trata de um código de computador, além disso essa tag faz com que as letras fiquem monoespaçadas facilitando a leitura do código.
Já a tag <pre> é usada para que o código seja visto da forma que ele foi digitado.

Exemplo

        
            num = int(input('Digite um número'))
            if num % 2 ==0:
               print(f'O número {num} é par')
               else:
               print(f'O número {num} é ímpar')
               print('FIM DO PROGRAMA')

Citações simples

Quando quisermos fazer uma citação simples usamos a tag <q>.
Exemplo: Como a famosa frase de Sócrates: Só sei que nada sei

Citações completas

Podemos criar também citações mais longas( em bloco), nesse caso usamos a tag <blockquote>.Podemos também colocar o link para o texto original, usando o parâmetro cite dentro da tag, apesar do link não ser visual o sites de buscas irão saber dele.

Abreviações

Sempre que quisermos deixa claro o significado da sigla para as pessoas que acessam e para o mecanismo de busca podemos usar a tag <abbr>.
Exemplo: O LOL é um jogo criado pela Riot Games.
