As Principais formatações

Negrito/Destaque

Quando usamos a tag <b> as palavras ficam em negrito, mas essa tag não é semântica.

Se utilizarmos a tag <strong> essas palavras ficam em destaque, essa é uma tag semântica.

Itálico/Ênfase

Se usar a tag <i> as palavra ficaram em itálico, não é uma tag semântica.

Já se colocarmos a tag <em> as palavras ficara em ênfase, essa é uma tag semântica.

Texto marcado

Se quisermos marcar um trecho do texto como marcador utilizamos a tag <mark>.

Texto grande e pequeno

A tag <big> era usado para que o texto ficasse maior, mas hoje em dia não é utilizado

A tag <small> é utilizada para que o texto fique menor

Texto deletado

A tag <del> faz que o texto possa ser lido, mas não é considerado.

Texto inserido

Se usar a tag <ins> o texto que está dentro dessa tag faz você prestar atenção no que tá escrito, porque fica uma linha por baixo da escrita .

Antes era utilizado a tag <u> (não semântica) que sublinhava o texto.

Texto sobrescrito

A tag <sup> é usada para escrever coisas semelhantes a x7+29

Texto subscrito

A tag <sub> é usada para escrever coisas semelhantes a H2O
