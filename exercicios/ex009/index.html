Trabalhando com Listas

Listas Ordenadas

Para criar um lista ordenada devemos usar a tag <ol> para demilitar a lista e a tag <li> que identifica cada item da lista.
Exemplo:

  1. Acordar Cedo
  2. Beber Água
  3. Escovar os Dentes
  4. Estudar

Para mudar o tipo do marcado usamos o parâmentro type dentro da tag <ol>. Temos essas opções:

  • 1- é padrão
  • A- lista alfabéticas em maiusculas;
  • a- lista alfabéticas em minúsculas;
  • I- lista com algarismos romanos em maiusculas;
  • i- lista com algarismos romanos em minúsculas.

Lista não Ordenadas

Para criar lista não ordenada usamos a tag <ul> para demilitar a lista e a tag <li> para criar cada um dos itens da lista.

  • Almoçar
  • Montar uma mesa
  • Compras

Assim como na lista ordenada usamos o type para mudar o marcador.
Temos essas opções:

  • disc- Padrão. Uma bola preta totalmente pintanda.
  • circle- Uma bola com uma borda preta mas sem preenchimento
  • square- Um pequeno quadrado preto
